# dimitrmo/cargo-verison-action

Uses [cargo-verison](https://github.com/dimitrmo/cargo-verison) to bump the version, tag and commit.

## Usage

```yaml
  - name: Cargo Verison
    uses: dimitrmo/cargo-verison-action@v1.0.7
```
